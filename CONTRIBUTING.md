# Contributing Guidelines

## How to Contribute

1. **Fork the repository**.
2. **Edit `data.json`** by adding/updating a company.
3. **Follow the format**:
   ```json
   {
     "name": "Company Name",
     "replies_to_email": "Yes/No",
     "pays_for_internship": "Yes/No",
     "website": "https://example.com"
   }
   ```
